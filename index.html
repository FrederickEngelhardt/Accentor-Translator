Accentor Tutorial:

1. Select your starting language using the menu. Default is English.

2. Select either an accent or a language to translate to using the menu.

Note:
1. Accents API has a limit of 5 queries/hr per IP address. Use a VPN if you want to use this API more.

2. Accents will always translate from English to Accented English. 3. All language choices are stored in local storage.
